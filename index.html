L'idée derrière le Brown Bag Lunch

Le Brown Bag Lunch est initiative née aux Etats-Unis. L'idée? Partager des connaissances durant un temps de midi. Chacun apporte son lunch et les attendees se retrouvent dans lieu convenu d'avance (entreprise, école, salle...) pour partager les connaissances sur un sujet spécifique à l'IT. Depuis le temps, l'idée a pris le large pour atteindre la France. C'est pourquoi, l'idée nous est venue de créer cette même communauté à Luxembourg. Nous désirons créer une communauté de speakers et de lieux à Luxembourg dans le cadre du Brown Bag Lunch Luxembourg.

Je veux créer un Brown Bag Lunch !

Alors? Convaincu par le concept? So let's go! Pour créer un Brown Bag Lunch il vous faut définir différent éléménts pour communiquer dessus: un lieu, un sujet, un speaker et une date. C'est pouruqoi, pour vous aider, nous avons créer ce site. Vous aurez l'occasion de consulter nos speakers ou les lieux disponibles

Je veux être speaker!

Super! Merci beaucoup! Pour ce faire, vous pouvez ajouter votre profile dans le fichier speaker.json. Il vous suffit de modifier la page js/speakers.js soit en forkant le projet et en soumettant un pull request, soit directement depuis Github via le bouton "edit".

Je veux acceuillir!

Un énorme merci! Pour ce faire, vous pouvez ajouter votre location dans le fichier location.json. Il vous suffit de modifier la page js/location.js soit en forkant le projet et en soumettant un pull request, soit directement depuis Github via le bouton "edit".
